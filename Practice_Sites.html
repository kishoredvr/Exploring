The Ultimate Legal Hacking Directory

An Exhaustive Master List of Cybersecurity Practice Environments

Virtual Machine Readiness

Many of the targets below (especially in the AD, IoT, and Self-Hosted sections) are explicitly designed to be downloaded and spun up in VirtualBox, VMware, or Docker. Always ensure you configure your hypervisor network settings (e.g., Host-Only or Internal Network) to prevent vulnerable targets from being exposed to the open internet.

1. OSINT & Reconnaissance

Platform URL Description
OSINT Dojo osintdojo.com Free, guided challenges ranging from basic image geolocation to tracing corporate infrastructure.
Sofia Santos' Exercises gralhix.com Massive repository of real-world OSINT challenges designed to train analysts in modern tooling.
TryHackMe (OSINT Path) tryhackme.com Dedicated guided rooms for learning Google Dorking, Shodan, and social media scraping.

2. Active Directory & Enterprise Networks

Platform Format Description
GOAD (Game of AD) GitHub Repo Fully dockerized, vulnerable AD lab. Perfect for practicing AS-REP Roasting, DCSync, and lateral movement.
DetectionLab GitHub Repo Vagrant/Packer environment that builds an AD network complete with Splunk, Zeek, and Suricata for Purple Teaming.
WazeHell's Vuln AD GitHub Repo A script-based AD setup designed to be deployed on a local Windows Server VM for practicing privilege escalation.

3. Web Apps & APIs (VM/Docker Targets)

Platform Source Description
OWASP Juice Shop GitHub Repo Arguably the most robust modern Javascript web target. Can be run instantly via Docker.
DVWA GitHub Repo Damn Vulnerable Web Application. Classic PHP/MySQL platform. Requires LAMP stack on local VM.
OWASP WebGoat GitHub Repo Java-based instructional framework featuring step-by-step modular explanations of security mistakes.

4. Cloud Infrastructure Security

Platform URL Description
flAWS & flAWS 2 flaws.cloud Interactive levels teaching common mistakes in AWS, focusing heavily on S3 buckets and IAM privileges.
SadCloud GitHub Repo Intentionally provisions insecure AWS infrastructure using Terraform to practice with tools like Prowler.
AzureGoat GitHub Repo Intentionally vulnerable Azure environment designed to practice Azure AD attacks and serverless exploitation.

5. IoT, Hardware & Firmware

Platform URL Description
Damn Vulnerable IoT GitHub Repo Framework simulating vulnerable IoT devices to test MQTT protocol attacks and default credential abuse.
IoTGoat GitHub Repo Vulnerable firmware based on OpenWrt. Download the image and use binwalk/QEMU on your VM to extract and emulate.
AttifyOS GitHub Repo A custom OS pre-configured with IoT targets and tools for practicing SDR, UART, and JTAG attacks.

6. Web3, Blockchain & Smart Contracts

Platform URL Description
Ethernaut ethernaut.com A Web3/Solidity-based wargame covering reentrancy attacks, integer overflows, and smart contract logic flaws.
Damn Vulnerable DeFi damnvulnerabledefi.xyz Specialized lab for learning offensive security of decentralized finance, covering flash loans and price oracle manipulation.
Capture the Ether capturetheether.com A game in which you hack Ethereum smart contracts to learn about their security architecture.
