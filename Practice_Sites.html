The Ultimate Legal Hacking Directory

An Exhaustive Master List of Cybersecurity Practice Environments

Strict Rules of Engagement

Every platform listed here is legally authorized for educational or testing use. However, restrictions vary. Some are live websites meant for web scanners, some require connecting to a closed VPN, and others must be hosted locally. Always check individual platform scopes before launching aggressive automated tools.

1. General Cyber Ranges & CTFs

Platform URL Description
TryHackMe tryhackme.com Guided learning paths, bite-sized rooms, and virtual networks. Highly recommended for beginners.
Hack The Box (HTB) hackthebox.com Live, unguided penetration testing labs, standalone boxes, and Active Directory pro-labs.
PicoCTF picoctf.org Permanent CTF track created by Carnegie Mellon University. Exceptional for foundational concepts.
Root-Me root-me.org Over 400+ challenges spanning web, network, forensics, automation, and system exploits.
CyberSec Labs cyberseclabs.co.uk Streamlined virtual lab environments specifically designed to practice core infrastructure testing.

2. Web App & API Security (Live Targets)

Platform URL Description
PortSwigger Academy portswigger.net/web-security The definitive web app training resource. Deep interactive labs focusing on server and client-side flaws.
Acunetix VulnWeb vulnweb.com Includes testphp, testasp, and testaspnet. Built explicitly for testing automated web vulnerability scanners.
Google Gruyere google-gruyere.appspot.com A sandboxed web testing application mimicking a microblogging tool filled with core flaws.
Hacker101 CTF ctf.hacker101.com Web-based CTF modules ranging from trivial to extremely complex to build bug bounty intuition.
Defend The Web defendtheweb.net Interactive security playground featuring dozens of text/code manipulation web levels.

3. Web Apps & APIs (Self-Hosted/Docker)

Platform URL (GitHub/Docs) Description
OWASP Juice Shop github.com/juice-shop Arguably the most robust modern Javascript web target. Contains a massive variety of hidden flags.
DVWA github.com/digininja/DVWA Damn Vulnerable Web Application. Classic PHP/MySQL platform with togglable difficulty levels.
OWASP WebGoat github.com/WebGoat Java-based instructional framework featuring step-by-step modular explanations of security mistakes.
DVGA (GraphQL) github.com/dolevd/DVGA Damn Vulnerable GraphQL Application. Tailored specifically to learn GraphQL injection and abuse.

4. Wargames & Command Line

Platform URL Description
OverTheWire (Bandit) overthewire.org The gold standard for beginners. Teaches critical Linux navigation, regex, and SSH.
UnderTheWire underthewire.tech Linear interactive SSH challenges focused heavily on Microsoft PowerShell mechanics and AD.

Platform URL Description
